Actualize Capstone Project

NEXT is an app that helps users plan for their next vacation. I love to travel, so I built a fullstack app to help plan for upcoming trips. The app allows users to search for sights and restaurants in the area they plan on traveling to, and also create an itinerary. Users are able to add sights and restaurants directly to their scheduled itinerary, and adjust each item on their itinerary.

By utilizing a MapBox API, the user can map out their itinerary day by day. The MapBox API calculates the distance and steps that are taken by traveling to each location in the day, and provides walking directions from one location to the next. Built on a Ruby-on-Rails backend with a Vue.js frontend, incorporating and modifying Bootstrap elements and Javascript transition elements.
