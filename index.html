Actualize Capstone Project

Towards the end of the coding bootcamp, I built a fullstack app called NEXT, combining all of the skills that I learned at Actualize. NEXT is a travel app that helps users plan for their next vacation. The app allows users to search for sights and restaurants in the area they plan on traveling to, and also create an itinerary. Users are able to add attractions directly to their scheduled itinerary. Once the attractions are on their itinerary, the user can adjust the date and time.

By utilizing a MapBox API, the user can map out their itinerary day by day. The MapBox API calculates the distance and steps that are taken by traveling to each location in the day, and provides walking directions from one location to the next. Built on a Ruby-on-Rails backend with a Vue.js frontend, incorporating and modifying Bootstrap elements and Javascript transition elements.
